# Introducere în Python 🐍

Acest depozit conține exerciții practice și concepte de bază scrise în limbajul Python, structurate pe sesiuni de învățare. Proiectul include exemple despre variabile, tipuri de date, operatori, funcții și structuri de decizie.

## 📂 Structura Proiectului

*   **Cap.1 Introducere:** Conține noțiunile fundamentale (Variabile, Tipuri de date, Operatori, Input).
*   **Sesiunea 2 Decizii:** Exerciții cu structuri condiționale (`if`, `elif`, `else`).
*   **Sesiunea 3 Bucle:** Exemple practice de utilizare a buclelor (`while`, `for`).
*   **Sesiunea 4 Colecții:** Lucrul cu liste, tupluri, seturi și dicționare.
*   **Sesiunea 5 Funcții:** Definirea funcțiilor, utilizarea `*args`, `**kwargs` și a funcțiilor lambda.

## 🚀 Cum se rulează codul

1. Asigură-te că ai Python instalat pe sistemul tău.
2. Deschide terminalul în folderul corespunzător.
3. Rulează orice script folosind comanda:
   ```bash
   python nume_fisier.py
   ```

## 🧠 Concepte Exersate
*   Interpolarea șirurilor de caractere (f-strings).
*   Operații matematice și incrementarea variabilelor.
*   Manipularea datelor introduse de utilizator.
